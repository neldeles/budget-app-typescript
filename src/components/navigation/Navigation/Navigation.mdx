Functionally acts as a wrapper for `NavItem` and `NavItemSubMenu`.

<Primary />

## Usage

`Navigation` is only to be used with the `NavItem` or `NavItemSubmenu` components.

### Imports

You only need to import the `Navigation` component. `NavItem` and `NavItemSubmenu` are
accessible as properties.

```jsx
import { Navigation } from "components/molecules/Navigation";
```

### Sample usage

```jsx
<Navigation>
  <Navigation.Item {...props} />
  <Navigation.ItemSubmenu {...props} />
</Navigation>
```
